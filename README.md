# Gestão MPE (Protótipo)

Protótipo **mobile-first** e **offline** para gestão financeira de micro e pequenas empresas.

## Como rodar
Basta abrir o arquivo `index.html` no navegador.

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub (ex: `gestao-mpe`).
2. Envie estes arquivos para a branch `main`.
3. Vá em **Settings → Pages** e selecione:
   - Source: `Deploy from a branch`
   - Branch: `main` / `/ (root)`
4. Aguarde e acesse o link gerado.
